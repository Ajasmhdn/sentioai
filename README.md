# 🎭 Sentio AI – Frontend  A modern, responsive frontend for **Sentio AI**, a deep learning-based system that detects human emotions from facial images.  Built with a clean UI and seamless user experience, this interface allows users to upload an image and instantly receive emotion predictions powered by a ResNet50 model.  ---  ## 🚀 Features  * 📤 Upload facial images easily * 🧠 Real-time emotion prediction via API * 🖼️ Image preview before submission * 🎯 Clean modal displaying predicted emotion * 🌙 Dark/light theme support * ⚡ Fast and responsive UI  ---  ## 🧩 Tech Stack  * **Next.js / React** * **NextUI** * **Axios** * **Tailwind CSS** * **FontAwesome Icons**  ---  ## 🔗 Backend Integration  This frontend connects to a Flask-based API:  ```bash http://127.0.0.1:5000/predict ```  Expected response format:  ```json {   "emotion": "Happy" } ```  ---  ## 🖥️ How It Works  1. Upload a facial image 2. Image is sent to backend API 3. Deep learning model processes it 4. Predicted emotion is returned 5. Result is displayed in a modal  ---  ## 📸 Preview  > Upload → Analyze → Emotion displayed clearly  ---  ## 🧠 Project Context  This project is part of a **Full-Stack Deep Learning system**, including:  * CNN model (ResNet50) * Backend API (Flask) * Docker deployment (planned)  ---  ## 📌 Future Improvements  * Real-time webcam emotion detection * Multi-face detection support * Emotion probability visualization * Cloud deployment  ---  ## 👨‍💻 Author  Developed as part of a course-based AI project focused on building end-to-end deep learning applications.  ---
